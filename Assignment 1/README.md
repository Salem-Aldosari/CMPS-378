Assignment 1

How to Run Each Program
Using Visual Studio:

Open Visual Studio
Create new Console App for each problem
Copy the code from each .cs file
Press F5 to run

Problem Summaries
Problem 1: BMI Calculator

Gets name, age, gender, height (feet & inches), weight
Converts height to inches
Calculates BMI using formula
Determines BMI category (Severe Thinness to Obese Class III)
Displays personalized results

Problem 2: Birthday Date Meaning Generator

Gets month, day, year (2000-2023)
Uses switch statements for meanings
Uses while loop to allow multiple tries
Shows meanings for month, day, and year

Problem 3: Road Trip Budget Estimator

Calculates gas needed based on miles and MPG
Calculates hotel cost (days - 1)
Calculates food cost
Shows detailed budget breakdown

Problem 4: LADWP Utility Bill Calculator

Uses tiered rates for electricity (4 tiers)
Uses tiered rates for water (4 tiers)
Calculates bills based on usage
Shows detailed monthly bill

Problem 5: AT&T Fiber Internet Bill Calculator

5 speed options (300, 500, 1000, 2000, 5000 Mbps)
Optional static IP add-on ($15)
Calculates 9.5% tax
Shows detailed billing summary
