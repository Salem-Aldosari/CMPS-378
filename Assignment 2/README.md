How to Run
Visual Studio:

Open Visual Studio
Create new Console App
Copy code from each file
Press F5

What Each Program Does
Problem 1: Freelance Income Tracker

Asks for name and number of projects
Uses FOR LOOP to get each project info
Calculates total income
Finds highest earning project
Shows average income per project

Problem 2: Monthly Expense Tracker

Asks for name
Uses WHILE LOOP to get unlimited expenses
Stops when user types "done" or enters -1
Tracks expenses by category
Finds highest spending category
Shows total and average
