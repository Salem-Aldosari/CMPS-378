Taco Loco POS System

How to Run
Visual Studio:

Open Visual Studio
Create new Console App
Copy the code from TacoLocoPOS.cs
Press F5

What It Does

Shows menu
Gets customer name
Gets package choice (A, B, C, or D)
Asks about drink (Y or N)
Gets quantity
Calculates total
Shows receipt
Checks for free dessert
